# Minecraft-Related

Erzyh의 마인크래프트 관련 플러그인 및 스크립트 모음입니다.

## 구조

```
Minecraft-Related/
├── Plugins/
│   ├── Aestas/        - 커스텀 낚시 플러그인
│   ├── EconomyXTA/    - 경제 플러그인 (1.19.3)
│   └── Stock/         - 주식 플러그인 (1.20.4)
└── Skripts/
    ├── EasyWarp/      - 워프 관리 스크립트
    ├── DefaultItem/   - 기본템 지급 스크립트
    └── Crate/         - 랜덤박스 스크립트
```

## Plugins

| 이름 | 설명 | 버전 |
|------|------|------|
| Aestas | 마인크래프트 커스텀 낚시 플러그인 | - |
| EconomyXTA | 마인크래프트 경제 플러그인 | 1.19.3 |
| Stock | 마인크래프트 주식 플러그인 | 1.20.4 |

## Skripts

| 이름 | 설명 |
|------|------|
| EasyWarp | `/워프관리`, `/워프` 명령어로 워프 포인트 생성/삭제/이동 |
| DefaultItem | `/기본템` 명령어로 신규 유저에게 기본 아이템 지급 |
| Crate | `/랜덤박스` 명령어로 랜덤 박스 시스템 운영 |
